# ItemStack

## 方法

### `getItem()`

获取包装的 `ItemStack` 对象中的物品。

- 返回值：
  - 返回一个 [Item](item.md) 对象，表示 `ItemStack` 中的物品。

### `getDamage()`

获取包装的 `ItemStack` 对象的损伤值。

- 返回值：
  - `ItemStack` 对象的损伤值。

### `getCount()`

获取包装的 `ItemStack` 对象的堆叠数量。

- 返回值：
  - `ItemStack` 对象的堆叠数量。

 ### `getInternalName()`

获取包装的 `ItemStack` 对象的键名。

- 返回值：
  - `ItemStack` 对象的键名。
 
### `getName()`

获取包装的 `ItemStack` 对象的名字。

- 返回值：
  - `ItemStack` 对象的名字。
